# curl commands
```
curl --location --request POST http://$FI_EVENT_HOST/event/postAndVerify \
--header 'Content-Type: application/json' \
--header "identifier: $FI_IDENTIFIER" \
--data-raw '{
  "name": "SendOrderStatusAck",
  "entityId": "3786",
  "entityType": "ORDER"
}'
```
